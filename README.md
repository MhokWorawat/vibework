# Vibework

Fullstack Kanban Board Web App

## Features
- Login / Logout (JWT)
- Kanban Task Board (Coming Soon)
- Deployed with Railway + Vercel

## Setup

### Backend
- Create `.env` based on `.env.example`
- `npm install`
- `npm start`

### Frontend
- Create `.env` based on `.env.example`
- `npm install`
- `npm run dev`
